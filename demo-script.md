# Reflexion — Demo Script (~90 seconds, 6 scenes)

## Scene 1: The Problem (15s)
**Visual**: Split screen — left: agent failing, right: dashboard showing low score
**Narration**: "AI agents are powerful, but they're stuck. They repeat the same mistakes every session. They don't learn. They don't improve. What if an agent could look at its own performance and fix itself?"

## Scene 2: The Failure (15s)
**Visual**: Agent executing a research task, making mistakes
**Narration**: "Watch Reflexion attempt a research task. The agent uses Gemini 3 to reason and plan, but its search queries are too broad. The results are shallow. The quality score: 0.45 out of 1. Not good enough."

## Scene 3: The Reflection (20s)
**Visual**: Agent querying Phoenix MCP, analyzing traces
**Narration**: "Here's where Reflexion is different. Instead of moving on, it stops and reflects. Using Arize Phoenix MCP, it queries its own execution traces. It identifies the exact failure point: search queries are too generic. It generates a targeted improvement: use more specific, domain-aware queries."

## Scene 4: The Success (15s)
**Visual**: Same task, improved execution, higher score
**Narration**: "Now watch Reflexion attempt the same task with its new strategy. The search queries are specific. The results are detailed. The quality score jumps to 0.87. A 93% improvement. The agent learned."

## Scene 5: The Dashboard (15s)
**Visual**: Dashboard showing improvement trends, trace comparison
**Narration**: "The dashboard shows the full improvement journey. Quality scores trending up. Each improvement tracked and validated. Trace comparison shows exactly what changed. This isn't just observability — it's autonomous self-improvement."

## Scene 6: The Vision (10s)
**Visual**: Architecture diagram, sponsor badges
**Narration**: "Reflexion is built on Gemini 3, Google Cloud Agent Builder, and Arize Phoenix. Any agent can be wrapped with Reflexion to start self-improving. The future of AI isn't just intelligent — it's self-evolving."

## Total Duration: ~90 seconds
## Key Moments:
- **Wow moment**: Scene 3 — agent querying its own traces
- **Proof moment**: Scene 4 — quality score jump from 0.45 to 0.87
- **Architecture moment**: Scene 6 — sponsor integration highlight
