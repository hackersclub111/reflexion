# Google Cloud Rapid Agent Hackathon — Intelligence Brief

## Hackathon Overview
- **Name**: Google Cloud Rapid Agent Hackathon
- **Theme**: Building Agents for Real-World Challenges
- **URL**: https://rapid-agent.devpost.com/
- **Deadline**: June 11, 2026 @ 2:00pm PDT (5:00pm EDT)
- **Prize Pool**: $60,000 across 6 partner tracks
- **Participants**: 11,398
- **Organizer**: Google Cloud + Devpost

## Requirements
- **Build with**: Gemini 3 + Google Cloud Agent Builder
- **Integrate**: At least 1 partner MCP server
- **Submit**: Hosted URL + GitHub repo (public, open-source license) + ~3 min demo video + track selection
- **New**: Must be newly created during contest period (May 5 - June 11, 2026)

## Our Strategy: Arize Track
**Why Arize**: Best starter kit, most compelling demo mechanic (self-improving agent), less crowded than MongoDB/Elastic, strongest judge alignment.
